# Problem and Goals

## Problem
There's a day that you don't want to go out just for buying a little stuffs. This will be the case for everyone, and that's why we have delivery apps. But the problem is, most of the delivery apps are quite expensive and quite complicated to use.

## Goals
Make delivery service as cheap as possible, and make it as simple as possible to use. The main goal of this project is to create a delivery service that is affordable and easy to use for everyone.

## Scope
For this version, I focused on core task management features only:
- consumer, courier, merchant accounts
- task creation and editing
