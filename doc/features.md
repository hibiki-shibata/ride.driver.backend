# Features

## Authentication
Consumers, couriers, merchant can create accounts and log in securely. This allows each user to manage their own personal task list.

## Task Management
Consumer can create task, merchant can accept the task, courier can update the status to make it be delivered.

## Account Management
Consumers, couriers, and merchants can manage/retrieve their accounts, including registration, data retrieval, and authentication. This allows for a personalized experience for each user type.

## Merchant Menu Management
Merchants can manage their menu items, including adding new items, updating existing ones, and removing items that are no longer available. This allows merchants to keep their offerings up to date for consumers.
