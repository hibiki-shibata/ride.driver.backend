# Overview

A backend service for a Ride app, built with Spring Boot and Kotlin. It provides APIs for managing consumers, couriers, delivery tasks, and authentication.
The backend handles business logic and data persistence, while the frontend (built with React) focuses on user experience. PostgreSQL is used for data storage, with JPA/Hibernate for ORM and Flyway for database migrations. This architecture allows for a clear separation of concerns, making the application easier to maintain and scale.
